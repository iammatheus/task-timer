# Task Timer
Projeto construído a partir do curso Formação React da Rocketseat<br /><br />
Melhorias feitas: 
 - Inclusão de notificações ao finalizar tarefa
   - Mensagem
   - Som

### Tecnologias utilizadas
- react
- typescript
- vite      
- zod
- hookform
- react-router
- reducer, immer
- styled-components
- eslint para padronização de código
